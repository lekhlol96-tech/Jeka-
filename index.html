КЛАССИФИКАЦИЯ МОТОРОВ LANCER X ПО СЕМЕЙСТВУ МЯКИШЕВЫХ

STAGE 0 // СТОК

Ржаной Диетический

Базовый мотор 1.5 литра. Едет медленно, зато полезно для здоровья. Навевает грусть, но Жека держится.

STAGE 1 // АЭРОДИНАМИКА

Воздушная Чиабатта

Облегченный кузов. Из-за дырочек в тесте вес Лансера снижается на 200 кг. Карбон курит в сторонке по сравнению с хрустящей корочкой.

STAGE 2 // СУРОВАЯ КЛАССИКА

Бородинский Чугун

Тот самый легендарный 4G63Т. Тяжелый, черный, с ароматом кориандра. Выдерживает давление до 3-х атмосфер и заезд до Ашана.

STAGE 3 // СТАНС И СТИЛЬ

Тостовый Квадрат

Идеальная квадратная геометрия. Машина ложится «пузом на асфальт» (дроп подвески). Жека нарезает углы так же ровно, как тостер режет хлеб.

FACTORY EDITION // ХАРДКОР

Дарницкий Заводской

Суровый обвес прямиком со старого хлебозавода. Обладает прочностью кирпича. Если такой Лансер врежется в столб — столб попросит прощения.

STAGE 4 // СВЕРХВЫХЛОП

8 ЗЛАКОВ (Twin-Turbo)

Максимальный гоночный спек. Восемь цилиндров, восемь злаков, полный привод Super All-Wheel Control. Заряжает Жеку энергией на 800 лошадиных сил.

СКАНИРОВАНИЕ ТЮНИНГА

Загрузи новое фото, и бортовой компьютер Лансера автоматически добавит его в базу данных Ralliart.

BORODINSKY RALLIART

Жека на низком старте

Lancer X прогревает резину. В баке плещется высокооктановый квас, на пассажирском сиденье ждет своего часа Дарницкий кирпич.

CHIABATTA STYLE

Итальянский выхлоп

Чиабатта дает идеальный продув выхлопной трассы. Звучит как породистый спорткар на перегазовке.
